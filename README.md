# palavrix

**https://pacbtz.github.io/palavrix/**

## Como jogar

Descobre a palavra oculta em 6 tentativas. Após cada tentativa, as cores das letras indicam:

- **Verde** — letra certa no lugar certo
- **Amarelo** — letra certa no lugar errado
- **Cinzento** — letra não existe na palavra

## URLs

Por defeito o jogo usa palavras de 5 letras com a palavra do dia de hoje.

### Comprimento da palavra

Adiciona `#N` ao URL para jogar com palavras de N letras (entre 5 e 11):

```
https://pacbtz.github.io/palavrix/#6
https://pacbtz.github.io/palavrix/#7
```

### Data específica

Adiciona `#N/YYYYMMDD` para combinar comprimento e data:

```
https://pacbtz.github.io/palavrix/#6/20260309
```

Para jogar com 5 letras numa data específica:

```
https://pacbtz.github.io/palavrix/#5/20260309
```

> A palavra é a mesma para todos os jogadores no mesmo dia e comprimento.
