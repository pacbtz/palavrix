# palavra

Jogo da palavra - Wordle game that always uses the word **PEDRO**.

## Play Online

**https://pacbtz.github.io/palavrix/**

## How to Play

- You have **6 attempts** to guess the 5-letter word.
- After each guess, tiles change colour to show how close your guess was:
  - **Green** — correct letter in the correct position
  - **Yellow** — correct letter in the wrong position
  - **Grey** — letter not in the word

## Development

The game is a single-file static HTML/CSS/JavaScript application deployed automatically to GitHub Pages via GitHub Actions on every push to `main`/`master`.
